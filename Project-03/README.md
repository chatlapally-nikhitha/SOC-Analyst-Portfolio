# Project 3 – Malware Hash & File Reputation Analysis

## Overview
This project demonstrates how a Security Operations Center (SOC) analyst investigates a suspicious file by analyzing its SHA-256 hash using VirusTotal. The goal is to determine whether the file is malicious and document the findings.

---

## Objective
- Investigate a suspicious file hash.
- Verify the file's reputation using VirusTotal.
- Identify indicators of compromise (IOCs).
- Produce a professional investigation report.

---

## Tools Used
- VirusTotal

---

## Investigation Workflow

1. Obtained the suspicious file hash.
2. Submitted the hash to VirusTotal.
3. Reviewed the detection ratio.
4. Examined file details and threat labels.
5. Recorded findings.
6. Created an investigation report.

---

## Skills Demonstrated
- Malware Analysis
- Threat Intelligence
- IOC Analysis
- Security Documentation
- Incident Investigation
