# Project 2 – Phishing Email Investigation

## Overview

This project demonstrates how a Security Operations Center (SOC) analyst investigates a suspicious phishing email by analyzing email indicators, URLs, IP addresses, and threat intelligence to determine whether the email is malicious.

---

## Objective

- Analyze a suspicious phishing email.
- Extract indicators of compromise (IOCs).
- Verify URLs and IP addresses using threat intelligence tools.
- Document the investigation findings.

---

## Tools Used

- VirusTotal
- AbuseIPDB
- Microsoft Outlook

---

## Investigation Workflow

1. Received a suspicious email.
2. Examined the email content and sender details.
3. Extracted the IP address and URLs.
4. Checked the IP reputation using AbuseIPDB.
5. Verified URLs using VirusTotal.
6. Documented the investigation results.
